# MCP Tools Reference

## Product Tools

| Tool | Description |
|------|-------------|
| `search_products` | Search products by query with pagination |
| `get_product` | Get detailed product information by ID |
| `list_brands` | List all available brands |
| `list_categories` | List category tree |
| `list_catalogs` | List all catalogs |
